# Campus Vibe 🎓📢

Campus Vibe is a smart digital notice board system designed to simplify communication between colleges and students by automatically extracting and organizing information from official college emails.

Instead of students manually checking multiple emails and notices, Campus Vibe provides a centralized platform where important announcements are displayed in an organized and easy-to-access manner.

---

## Features

- 📧 Automatic extraction of notices from official college emails
- 📝 NLP-based extraction of important details such as:
  - Event Name
  - Date
  - Time
  - Venue
- 🔍 Categorized notices for easier navigation
- 👤 User authentication and authorization
- 📱 Responsive user interface
- 📢 Centralized digital notice board for students
